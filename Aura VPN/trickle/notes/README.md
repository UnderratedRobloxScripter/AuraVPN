# Aura VPN Landing Page

A minimalist, dark-themed landing page for the Aura VPN project.

## Features
- **Design**: Acrylic/Glassmorphism UI with dark theme.
- **Components**: Hero section, Feature grid, Responsive Navbar.
- **Stack**: React (via CDN), TailwindCSS.

## Project Structure
- `index.html`: Entry point, styles, and CDN links.
- `app.js`: Main React App component.
- `components/`: UI components (GlassCard, Navbar, Hero, etc.).

## Customization
- Theme variables are defined in `index.html` under `@layer theme`.
- To change the accent color, modify `--primary-glow`.
